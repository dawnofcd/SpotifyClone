🎧 Spotify Clone – React App
A basic Spotify UI clone built with React, styled using Tailwind CSS, and state managed via Context API.

📚 Tech Stack
⚛️ React JS

🎨 Tailwind CSS

📦 React Context API

🔊 React H5 Audio Player

⭐ FontAwesome Icons

🚀 Getting Started
1. Clone the repository
bash
Copy
Edit
